#cli for upyun
使用python开发的upyun 的一个shell，支持命令行补齐。

#功能

1. 设置bucket, username, passwd；  
2. 命令行帮助；  
3. 命令行使用<tab> 补齐 ；
4. 自己开发的python sdk，功能类似官方的sdk
